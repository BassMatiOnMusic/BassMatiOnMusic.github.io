These are all major triads, with a root, a major 3rd and a perfect fifth interval. You may notice that real tunes also use minor and diminished triads, as well as dominant 7th chords. These chord types can be derived from the major triads, but for simplicity, we will stick to the major triads, even if that is harmonically limiting…

Now let's find all the Cs on the Fretboard. You need the following navigation rules:

  1. The next C one octave down is +3 frets +3 strings away.
  2. The next C one octave up is +2 frets −2 strings away.
  3. If you crossed from string 3 to string 2, add one fret to the formula.
  4. If you crossed from string 2 to string 3, subtract one fret from the formula.
  5. The C on string 6 can be found two octaves higher on the 1st string. Rules 3 and 4 do not apply for this rule.

To get the the C on string 6, fret 8, apply rule 1. To get to the C on string 1, fret 8, apply rule 5. To get to string 4, fret 10, apply rules 1 and 4. To get to string 2, fret 13, apply rules 2 and 3. And so on.

Summary: The trick is to pick a note (C) anywhere on the fretboard, declare it as the root of the chord, and then find a CAGED shape that has a root on that string. Despite the name of the shape, we allways got C chords. And of course, we are not limited to C notes. Any other note works well, in the same way. So, one of the most important things is to know where the roots are in each shape.

The benefit is, umm, yeah, the benefit… Let me think…

Admittedly, I have no idea what the damned benefit of this damned CAGED system is! Basically, we are applying some chord shape on the fretboard to get a specific chord with a specific root note. The procedure is the same for every other chord shape.

The CAGED shapes are moveable? Well, all chord shapes are movable, within the limitations of the left hand, of course. And the G shape from above for example is hard to move. Modify the shape to a Gm triad shape, and it is impossible to fret it down!

Now, the world doesn't consist of only major triads. There are also minor triads. A minor triad can be derived from a major triad by lowering the major 3rd by a half-step to a minor 3rd. So in addition to the location of roots, you must learn the location of the 3rds, too. And there are a lot of 3rds in the shapes!

Then there are diminished triads. They can be derived from a minor triad by lowering the perfect fifth by a half-step to a diminished fifth. So you also need to learn the position of the 5th. And there are a lot of fifths in the shapes!

So, there are only five shapes to learn that can be applied and tweaked, right? Wrong. There are very much more! Long story short: CAGED is not only difficult, is also limiting!

What we really need is some sort of clever reduction of knowledge to a bare minimum, yet complete but not limiting. In fact, the student can safely focus on a set of three major triad shapes on a set of three adjacent strings, from which all other shapes and chord types can easily be derived.
