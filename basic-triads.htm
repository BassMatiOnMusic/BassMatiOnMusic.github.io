It is a major triad (R, 3, 5) in root position, i. e. the root note is at the bottom, and the 5th interval is on the top.

Note that there is no note information here – note names (i. e. pitch values) are irrelevant for the chord quality. A major triad is always a major triad, no matter where it is played. It is only when you are putting a chord shape into a song context, then you need to locate the root note of the shape on a specific note, and layout the rest of the pattern in relation to the root. And that is simply a matter of practice.

Level 2 – Triad Inversions

On the way to a proficient guitarist, you need to know all three inversions of a major triad.

Inversions

Quick reminder: A chord in the root position is a stack of third intervals with the root in the bass: (R, 3, 5). For example, a G triad, (G, B, D). If you move the top note (D) an octave down, it is still the same chord, but now the D is in the bass (D, G, B). This is the first inversion. If you do it again, you get the B in the bass: (B, D, G). This is the second inversion. The chord color changes, but the chord quality remains – a G triad.

Note the stack structures in the staff: The root position has all three notes close together. The first inversion has two adjacent notes on the top, whereas the 2nd inversion has two adjacent noted at the bottom.

Now we will focus on the string set (➁ ➂ ➃), and forget about the note names.

You should still be able to recognize the note stack structures in the score, they didn't change, only the notes have different pitch values. This has advantages in the tabulature – we can now focus on the area around the fourth fret.

Developing the Triad Inversions

To continue on the bold statement from above: We can develop the triad inversions from the triad in root position easily by applying an octave rule and two border crossing rules:

  1. To move a note down an octave, add three strings and three frets to the current position.
  2. If you cross the border between strings 2 and 3 down, subtract one fret from the result.
  3. If a pattern is shifted up vertically by one string, string 3 must be shifted one fret to the right when it lands on string 2.

So let's check it out!

At the end of level 2 we have three chord shapes to start your fretboard journey with. You should already be familiar with shifting chord shapes horizontally. In level 3 we are going to shift these patterns vertically.

Level 3 – Vertical Shifting

Vertical Shifting Rules

We already used the vertical shifting rule above, but it can't hurt to repeat:

If the guitar was tuned in all fourths, that's all we had to memorize. But (un)fortunately, the tuning interval between strings 2 and 3 is a major 3rd (provided a six string guitar in standard tuning). This has an effect on the shapes when they are shifted up and down vertically:

  1. If a shape row crosses the border between the 3rd and 2nd string upwards, the row has to be shifted one fret to the right.
  2. If a shape row crosses the border between the 2nd and 3rd string downwards, the row has to be shifted one fret to the left.

These border crossing rules ensure that the intervals between the notes in the shape remain constant. In other words, the chord remains intact, only the pitch values change.

Level 4 – Transformations

Doubling of Notes

Notes can be doubled to "fatten" the sound of a triad, and eventually arrive at the well-known "beginner chord" shapes, some of which employ opens strings. This demonstrates that learning many chords can be avoided to a great deal, and instead leaves us with better fretboard knowledge, because we know the structure of our chords, not only their name, shape and location.

Minor Chord Transformation

If we change the major third in a major triad into a minor third, we get a minor triad with the structure (R, m3, p5).
